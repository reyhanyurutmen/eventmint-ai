# EventMint AI 🚀

EventMint AI is a simple web app that turns event participation into collectible digital badges.

## ✨ Features
- Generate personalized event badges
- Download badge as PNG
- NFT-style metadata (JSON export)
- Clean and simple UI

## 🧠 Idea
This project explores how events can create digital identity and collectible experiences using Web3 concepts.

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🔮 Next Steps
- Wallet connection
- On-chain minting
- QR code badge claiming

## 💡 Why this matters
Events, communities, and creators can use this to create digital collectibles and identity systems.
